Projects

Happy Camper

An Alexa skill designed to use the ACTIVE API to obtain campground information. Happy Camper is available at the Amazon Skills Store . You can ask Happy Camper for things such as a campground's description, amenities, pet policy, location and nearby attractions. You can also get a list of campgrounds near a certain location such as a city, park, or landmark. Happy Camper was built using Node.js and hosted on AWS Lambda

Rflct

A smart mirror that uses the Kairos API to recognize then load user profiles. We use Amazon Alexa to answer any questions you may want to ask. Some of the modules that can be loaded to an individual’s profile include current weather, weakly forecast, traffic for your commute, news headlines, time, and inspirational quotes.

Eclair

Health 2.0 Code-A-Thon, is a 24 hour contest to develop an application to help seniors manage their medical. Eclair is my teams submission with the Pokitdok API we help to calculate when individuals with medicare part-d will hit the "donut hole" (an insurance coverage gap that requires people to 45% cost of medication out of pocket) and suggest alternative medications using First Databank API, that will reduce the cost.
